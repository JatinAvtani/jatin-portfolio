# Jatin Avtani - Personal Portfolio

Welcome to my personal portfolio! This is a modern, highly optimized single-page web application built to showcase my expertise as a Full-Stack Software Engineer and GenAI Enthusiast.

## Tech Stack
- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Animations**: Custom Tailwind keyframes

## Features
- **Dark Mode Aesthetic**: A sleek, futuristic dark theme with glassmorphic elements and glowing text.
- **Fully Responsive**: Perfectly scaled for mobile, tablet, and desktop devices.
- **Live Contact Form**: Working backend integration via FormSubmit AJAX.
- **Real-Time Hit Counter**: Automatically tracks live global page views using CounterAPI.

## Getting Started

To run this project locally, simply clone the repository and run the following commands:

```bash
npm install
npm run dev
```

Visit `http://localhost:5173` to view the live site.
